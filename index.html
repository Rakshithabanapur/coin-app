CLI Tool: coin

Linux installation

Commands to download and execute the application:
Step 1 → echo "deb [trusted=yes] https://rakshithabanapur.github.io/coin-app stable main" | sudo tee /etc/apt/sources.list.d/coin-app.list
Step 2 → sudo apt update
Step 3 → sudo apt install coin

MacOs installation

Commands to download and execute the application:
Step 1 → brew update
Step 2 → brew tap rakshithabanapur/coin
Step 3 → brew install coin
